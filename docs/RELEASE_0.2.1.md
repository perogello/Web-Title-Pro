# Web Title Pro 0.2.1

## RU

- Упрощен `Yandex Disk` flow: оставлен один понятный режим `Public Link`
- Улучшены `Data Source` и `Mapping` для локальных и `vMix` титров
- Добавлен импорт локальных шаблонов из `ZIP` и папки
- Загруженный шаблон теперь можно сразу добавить в rundown
- Добавлено удаление кастомных шаблонов
- Для `khural` добавлен редактор шрифтов, размеров и цветов по полям
- Системные шрифты Windows теперь доступны в редакторе
- Добавлена кнопка открытия папки шаблона из списка титров
- Улучшена стабильность portable updater и общая чистка кода

Рекомендуемый файл для запуска:

- `WebTitlePro.exe`

## EN

- Simplified the `Yandex Disk` flow to one clear `Public Link` mode
- Improved `Data Source` and `Mapping` for local and `vMix` titles
- Added local template import from `ZIP` and folders
- Uploaded templates can now be added to the rundown immediately
- Added custom template removal
- Added per-field font, size, and color editing for `khural`
- Windows system fonts are now available in the style editor
- Added a button to open a title template folder from the titles list
- Improved portable updater stability and continued code cleanup

Recommended file to launch:

- `WebTitlePro.exe`
