# Web Title Pro 0.3.4

## RU

- Исправлен внешний вид биндов в Shortcuts и MIDI: строки теперь одинаковой компактной ширины без пустой растянутой области справа.
- Learn-состояние больше не расширяет строку бинда: текст сокращен до компактного вида.
- MIDI больше не создает дефолтные бинды `Any device / Note 60-64`: команды без назначения показываются как `Not assigned`.
- Исправлена настройка таймера во время работы: текущие стрелки теперь добавляют или убавляют время одним кликом без необходимости многократно перекликивать.

## EN

- Fixed Shortcuts and MIDI binding layout: rows now use one compact consistent width without the stretched empty area on the right.
- Learn state no longer expands the binding row: the prompt text is now compact.
- MIDI no longer creates default `Any device / Note 60-64` bindings: unassigned commands now show `Not assigned`.
- Fixed live timer adjustments: the existing arrows now add or subtract time with one click while the timer is running.
