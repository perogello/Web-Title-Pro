# Web Title Pro 0.3.3

## RU

- Shortcuts / MIDI: компактная строка бинда без пустого растягивания вправо.
- Больше места под название команды, чтобы длинные названия реже переносились.
- `Not assigned` приведен к одному виду в Shortcuts и MIDI.
- Известная проблема: в Live Data Source таймер во время работы может не принимать ручное изменение секунд/минут/часов с первого клика. Исправлено в `0.3.4`.

## EN

- Shortcuts / MIDI: compact binding row without useless empty width on the right.
- More room for command names so long labels wrap less often.
- `Not assigned` styling is now consistent across Shortcuts and MIDI.
- Known issue: in Live Data Source, a running timer may not apply manual second/minute/hour adjustments on the first click. Fixed in `0.3.4`.
