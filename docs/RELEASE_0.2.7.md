# Web Title Pro 0.2.7

## RU

- Исправлен updater flow: приложение корректно запрашивает сохранение проекта и закрывается для установки обновления
- Добавлена кнопка `Update Now` в разделе `Updates`
- Возвращен `Control UI URL` в `Settings -> Output`
- Добавлены глобальные команды `Next Title` и `Previous Title` для keyboard, MIDI и Bitfocus
- Улучшен multi-operator workflow: выбранный `Output` теперь локален для каждого клиента
- Добавлены новые built-in шаблоны `Khural PIP Title` и `Khural PIP Title 2`

Рекомендуемый файл:
- `WebTitlePro.exe`

## EN

- Fixed the updater flow so the app now handles project save prompts and closes correctly for update installation
- Added an `Update Now` button to the `Updates` section
- Restored the `Control UI URL` in `Settings -> Output`
- Added global `Next Title` and `Previous Title` commands for keyboard, MIDI, and Bitfocus
- Improved the multi-operator workflow by making the selected `Output` local to each client
- Added new built-in templates: `Khural PIP Title` and `Khural PIP Title 2`

Recommended file:
